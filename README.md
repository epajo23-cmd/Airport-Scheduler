This project is an Airport Scheduling System developed as part of an AI course project done by Eden Pajo(github.com/epajo23-cmd) and Thomas Kroj(github.com/tthomas2513). The system simulates the scheduling and management of airport operations, focusing on efficient allocation of flights, runways, and time slots.

The application is designed to handle scheduling constraints such as avoiding conflicts between flights, optimizing runway usage, and ensuring smooth airport traffic flow. It incorporates algorithmic logic to assign and manage schedules in a structured and efficient manner.

The project follows a modular design, separating core logic, data handling, and system operations. It demonstrates problem-solving in constrained environments, where multiple entities must be coordinated without overlap.


Key aspects of the project include:
- Flight scheduling and time-slot allocation
- Conflict detection and resolution
- Efficient resource (runway) management
- Structured and maintainable code design

This project highlights the application of algorithmic thinking and basic AI concepts in solving real-world scheduling problems.

# Airport Scheduling System

An Airport Scheduling System developed as part of an AI course project. The system simulates airport operations by managing flight schedules, runway allocation, and time-slot organization while avoiding conflicts.

---

## Features

- Schedule flights with specific time slots
- Allocate runways efficiently
- Detect and prevent scheduling conflicts
- Manage multiple flights simultaneously
- Optimize airport traffic flow

---

## Project Focus

This project focuses on solving a **constraint-based scheduling problem**, where multiple flights must be assigned resources (runways and time slots) without overlap.

It demonstrates:
- Algorithmic problem solving
- Resource allocation strategies
- Conflict detection and resolution
- Structured system design
